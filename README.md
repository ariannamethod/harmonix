GPT-4 has 50,000 tokens. Knows everything. Says nothing interesting.  
HAiKU has 500 words. Knows nothing. Every word **hits**.

**Constraint doesn't limit. Constraint FOCUSES.**

### 2. Weightless Bookends (Circular Symmetry)
